# Чеклист EDA (Exploratory Data Analysis)

Практический и универсальный чеклист для анализа данных перед обучением
модели.

------------------------------------------------------------------------

## 1) Обзор датасета

-   Размер: количество строк и колонок
-   Просмотр: head / tail
-   Типы данных (числовые, категориальные, даты)
-   Используемая память

**Вопросы:** 
- ### Достаточно ли данных? 
- ### Типы соответствуют ожиданиям?

------------------------------------------------------------------------

## 2) Анализ target (целевой переменной)

-   Распределение (гистограмма)
-   Статистики (mean, median, std, квантили)
-   Перекос (skewness)
-   Выбросы

**Идеи:** 
- При сильном перекосе попробовать трансформации (например
логарифм)
- Определить метрику заранее (RMSE, MAE и т.д.)

------------------------------------------------------------------------

## 3) Пропуски (missing values)

-   Количество и доля NaN по колонкам
-   Структура пропусков

**Решения:** 
- Удалить колонку (если слишком много пропусков)
- Заполнить (mean / median / mode / константа) - Добавить флаг "был
пропуск"

------------------------------------------------------------------------

## 4) Типы признаков

Разделить признаки: - Числовые - Категориальные - Даты

**Действия:** 
- Привести типы к нужным 
- Продумать кодирование категорий

------------------------------------------------------------------------

## 5) Анализ отдельных признаков (univariate)

-   Числовые: распределение, выбросы
-   Категориальные: частоты значений

**Проверки:** 
- Константные признаки 
- Сильно перекошенные признаки

------------------------------------------------------------------------

## 6) Связь признаков с target

-   Числовые → target: scatter plot, корреляция
-   Категориальные → target: boxplot

**Цель:**
- Найти признаки с сигналом

------------------------------------------------------------------------

## 7) Связь признаков между собой

-   Корреляционная матрица
-   Поиск дублирующих признаков

------------------------------------------------------------------------

## 8) Выбросы

-   Визуально (scatter, boxplot)
-   Статистически (IQR, z-score)

**Решения:**
- Удалить / ограничить / оставить

------------------------------------------------------------------------

## 9) Дубликаты и качество данных

-   Проверка на дубликаты
-   Нереалистичные значения

------------------------------------------------------------------------

## 10) Data Leakage

-   Признаки, которые напрямую содержат target
-   Информация из будущего

------------------------------------------------------------------------

## 11) Валидация

-   K-Fold / Stratified / Time split
-   Избегать утечек

------------------------------------------------------------------------

## 12) Базовая модель (baseline)

-   Простая модель или среднее значение
-   Быстрая проверка пайплайна

------------------------------------------------------------------------
